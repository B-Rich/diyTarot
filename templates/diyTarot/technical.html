Technical Details

diyTarot is a Django application, so it is written in Python with a database backend (in my case, MySQL). The front end uses HTML interspersed with Django's template language, with CSS to handle layout and styling and JavaScript in some places to do asynchronous updating and certain effects.

I opted to use Django instead of coding everything myself or using another package because:

  1. It's Python, which is a joy to code in.
  2. It has an amazing automatic admin which makes it easy to start adding content immediately.
  3. Great documentation.

Once I finish adding the core features and do some basic portability and accessibility testing, I plan to make diyTarot available as an open source download, so that anyone else using Django can add customizable tarot card readings to their site as an application. The more tarot websites out there the better!
